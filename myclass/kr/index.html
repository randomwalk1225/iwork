기능 보기 도입 문의

PROBLEM

학원 행정, 아직도 엑셀로?

선불 학생은 남은 시간 계산, 후불 학생은 이번 달 출석 정산, 횟수제 학생은 남은 회차 확인 — 결제 방식마다 계산이 다릅니다.

엑셀로 관리하면 실수가 나고, 안내문 하나 보내려면 학생별로 따로 작성해야 합니다.

MyClass는 이 모든 것을 자동화합니다.

출석만 찍으면 결제 방식에 맞춰 회비가 자동 계산되고, 안내문이 생성되고, 매출 통계가 잡힙니다.

FEATURES

핵심 기능

출석 관리

학생별 출석 기록, 월별 목표시간 관리, 연간 출석 현황 추이 그래프.

회비 자동 계산

선불(잔여시간), 후불(출석 정산), 횟수제(남은 회차) 세 가지 방식 자동 처리.

안내문 생성

학생별 맞춤 회비 안내문 자동 생성. 전달 이력 관리, 납부 확인 체크.

매출 통계

결제방식별 매출 조회. 월별 매출 추이, 학생별 결제 이력 한눈에.

횟수제 현황

회차 기반 청구 관리. 남은 수업 횟수 자동 추적, 소진 알림.

데이터 처리

CSV 데이터 임포트, 검증, 리포트 자동 생성. 기존 데이터 마이그레이션 지원.

BILLING

3가지 결제 방식 지원

선불제

목표시간 선납 → 출석 차감 → 잔여시간 안내

후불제

월말 출석 정산 → 사용 시간 기준 청구

횟수제

회차 구매 → 수업 차감 → 소진 알림
