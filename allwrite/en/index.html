CORE

Real-Time Point-Shape Synchronization

At the heart of AllWrite is perfect 1:1 correspondence between points and shapes. Drag a segment's endpoint and the segment follows. Drag a circle's center and the whole circle moves. Create a midpoint — when either parent moves, the midpoint recalculates automatically. Every connected shape updates in real-time, smoothly, simultaneously.

Dependency Graph

Each shape tracks what it depends on. Change one point and every dependent shape cascades its update.

Smooth Animation

Transformations are instant but never jerky. All shapes respond with real-time animation while you drag.

Constraint Preservation

A rhombus stays a rhombus. A right triangle keeps its right angle. Geometric properties are preserved no matter how you drag.

GEOMETRY

20+ Geometry Tools

5 point types, 3 line types, 3 circle types, 8+ polygon types. Every shape is controlled by points.

Points (5)

Fixed, slider, midpoint, division point (m:n ratio), constrained (on line/circle)

Lines (3)

Segment, ray, infinite line. Drag endpoints for real-time transform.

Circles (3)

Center-radius, diameter, three-point circle. Drag to resize/reposition.

Polygons (8+)

Regular 3-12, rectangle, square, parallelogram, rhombus, kite, right triangle, free polygon

Special Polygons — geometric properties are always preserved

Rectangle

Defined by diagonal endpoints. 90° angles maintained no matter how you drag.

Rhombus

Defined by diagonal. All four sides always equal length.

Right Triangle

Defined by hypotenuse endpoints. The right angle is always preserved.

GRAPHING

Type an equation. See the graph instantly.

Beautiful LaTeX-rendered formulas via KaTeX, converted to graphs in real-time.

Cartesian

y = f(x)

y = x², y = sin(x), etc. Auto discontinuity detection.

Parametric

x(t), y(t)

x = cos(t), y = sin(t) and other parametric curves.

Polar

r(θ)

Spirals, rose curves. Auto Cartesian conversion.

Implicit

f(x, y) = 0

x² + y² = 1 and other implicit curves. Marching Squares algorithm.

Inequalities

y > f(x)

Shaded solution regions for visual understanding.

Auto Intersections

Detected & marked

Function intersections, x/y-intercepts found and displayed automatically.
