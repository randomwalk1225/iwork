FEATURES

핵심 기능

시간제한 시험

시험별 제한시간 설정. 시간 초과 시 자동 제출. 남은 시간 실시간 표시.

자동 저장

학생이 답을 입력할 때마다 자동 저장. 브라우저를 닫아도 답안 유지.

자동 채점

문제별 배점 기준으로 자동 점수 계산. 서술형은 채점자 수동 채점.

역할 기반 접근

관리자, 채점자, 학생 3가지 역할. 역할별 다른 기능과 화면.

시험 분류

시험 유형별 카테고리 분류. 과목, 단원, 난이도별 체계적 관리.

응시 이력

학생별 응시 기록, 점수 추이, 소요시간 분석. 관리자 대시보드.
